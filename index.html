El lenguaje de marcado de hipertexto (HTML) es un lenguaje informático que compone la mayoría de las páginas web y aplicaciones en línea. Un hipertexto es un texto que se utiliza para hacer referencia a otros piezas de texto, mientras que un lenguaje de marcado es una serie de marcas que le dice a los servidores web el estilo y estructura de un documento. HTML es muy simple de aprender y usar.
HTML
CSS significa hojas de estilo en cascada. Es el lenguaje para describir la presentación de páginas Web, incluidos los colores, el diseño y las fuentes, lo que hace que nuestras páginas web sean presentables para los usuarios. CSS está diseñado para hacer hojas de estilo para la web. Es independiente de HTML y se puede utilizar con cualquier lenguaje de marcado basado en XML.
CSS
JavaScript es un lenguaje de secuencias de comandos o programación que le permite implementar características complejas en la páginas web: en vez de mostrar la informacion de manera estatica, se puede observar: visualización de actualizaciones de contenido oportunas, mapas interactivos, gráficos 2D/3D animados, desplazamiento jukeboxes de video,etc.
